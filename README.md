# SMC — Truck Material Volume (live demo)

**Live demo → https://lumigrade.github.io/smc-truck-demo/**

Smart Material Control measures how much material a truck carries at the site gate,
automatically, from two cross-mounted SICK picoScan150 LiDAR scanners fused into a
single 3D model. This page shows a **real scan of truck 37H11558** — comparing the
**full** vs **empty** bed gives **~14.0 m³ (≈20.9 t)** of material, with the load
highlighted distinctly from the truck/bed.
